# Alliance Street Accountancy - Website

## Project Overview
A professional marketing website for Alliance Street Accountancy Ltd, a UK-based accounting firm. It is a modern, responsive single-page application (SPA) designed to showcase accounting services and generate leads.

## Tech Stack
- **Framework**: React 19 with TypeScript
- **Build Tool**: Vite 8
- **Styling**: Tailwind CSS 4 (via Vite plugin)
- **Animations**: Framer Motion + GSAP (with ScrollTrigger)
- **Video**: HLS.js for streaming video
- **Package Manager**: npm

## Project Structure
- `src/` — React source code (App.tsx, main.tsx, index.css, assets/)
- `public/` — Static assets (logo, favicon, PDF checklist)
- `scripts/` — Python script for PDF generation (`build-pdf.py`)
- `dist/` — Built output (generated by `npm run build`)

## Development
- **Dev server**: `npm run dev` (port 5000, host 0.0.0.0)
- **Build**: `npm run build`
- **Lint**: `npm run lint`

## Deployment
Configured as a **static** deployment:
- Build command: `npm run build`
- Public directory: `dist`
